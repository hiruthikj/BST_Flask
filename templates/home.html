What is a Binary Search Tree?

A binary search tree is a data structure that quickly allows us to maintain a sorted list of numbers. It is composed of nodes, which store data and also links to up to two other child nodes. It is the relationship between the leaves linked to and the linking leaf, also known as the parent node, which makes the binary tree such an efficient data structure.


Binary Search Tree is a binary tree data structure which has the following properties:

  1. The left subtree of a node contains only nodes with keys lesser than the node’s key.
  2. The right subtree of a node contains only nodes with keys greater than the node’s key.
  3. Each left and right subtree must also be a binary search tree.
