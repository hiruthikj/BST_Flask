Deletion :

There are three possible cases when deleting a node in a BST :

  1. Deleting a leaf node :

    2. Simply remove from the tree.


  2. Deleting a node with one child :

    3. Copy the child to the node and delete the child.


  3. Deleting a node with two children :

    4. Find inorder successor of the node. Copy inorder successor to the node and delete the inorder successor. Inorder predecessor can also be used.
